# E-commerce Customer Sentiment Analysis & Power BI Dashboard 📊

## 📌 Project Overview
This project analyzes customer feedback for a Women's Clothing E-commerce platform. Using Natural Language Processing (NLP) in Python, I classified over 21,000 customer reviews into Positive, Neutral, and Negative sentiments. Finally, I built an interactive, dark-themed Power BI dashboard to help business stakeholders make data-driven decisions based on real customer experience.

## 🛠️ Tech Stack & Tools Used
* **Python (Jupyter Notebook):** Data Cleaning, Text Preprocessing, and Sentiment Analysis (NLP).
* **Power BI:** Data Visualization, Interactive Dashboards, DAX queries, and Data Modeling.
* **Pandas & NumPy:** Data Manipulation.

## 🚀 Key Business Insights & Features
* **Sentiment Breakdown:** Analyzed the distribution of sentiments across different clothing departments (e.g., Tops, Dresses, Intimates).
* **Rating Correlation:** Mapped extracted text sentiment against actual customer ratings to identify discrepancies and genuine feedback.
* **Interactive Filtering:** Drill-down capabilities for managers to instantly filter reviews by sentiment and read exact customer comments to pinpoint root causes of dissatisfaction.
* **KPI Tracking:** Real-time metrics for Total Feedback count and Average Rating.

## 📂 Repository Structure
* `analysis.ipynb`: The Python script containing the entire NLP pipeline, from raw text processing to sentiment classification.
* `Womens Clothing E-Commerce Reviews.csv`: The original raw dataset.
* `cleaned_sentiment_data.csv`: The processed dataset with assigned sentiment labels, ready for visualization.
* `ecommerce_project.pbix`: The final Power BI dashboard file.
